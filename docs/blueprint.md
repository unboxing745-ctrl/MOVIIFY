# **App Name**: ReelFinder

## Core Features:

- Movie Search: Allows users to search for movies using keywords, titles, actors, and genres. Uses an external movie API (TMDB/OMDB).
- User Authentication: Enables users to create accounts, log in, and manage their profiles. Implemented with secure password storage.
- Movie Reviews and Ratings: Enables users to write, submit, and read reviews, as well as rate movies. Persist reviews in the mySQL database.
- Personalized Movie Recommendations: Recommends movies based on viewing history, ratings, reviews and preferred genres using collaborative filtering.
- Responsive Design: Ensures the website is fully responsive and accessible across different devices (desktops, tablets, smartphones).
- Review Sentiment Summarizer: AI tool that summarizes user reviews and extracts key opinions.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and a sense of reliability.
- Background color: Light gray (#ECEFF1), desaturated hue of the primary color to support readability and focus on content.
- Accent color: Light purple (#7E57C2) for interactive elements and highlights, providing a vibrant contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text
- Code font: 'Source Code Pro' for displaying code snippets.
- Use flat, minimalist icons in a single accent color to maintain a clean and modern aesthetic.
- Employ a grid-based layout to maintain a structured and organized appearance, ensuring content is easily scannable.
